---
description: Scaffold the Markdown Viewer Electron project from scratch
---

Set up the project with the following steps:

1. Initialize the Electron project using electron-forge with Vite + TypeScript template
2. Install all required dependencies
3. Set up the project directory structure as defined in DESIGN.md
4. Configure TypeScript, ESLint, and Vite
5. Create the basic main process with window creation
6. Create the preload script with contextBridge API
7. Set up the React renderer with basic App component
8. Verify the app launches successfully with `npm start`

Follow the conventions in CLAUDE.md strictly.
